# CSEB Mastery — Ghana Edition

A self-paced training application for launching a **Compressed Stabilized Earth Block (CSEB)** manufacturing business in Ghana.

**Live demo:** _Coming soon — update this link once GitHub Pages is enabled._

## What is this?

A complete training program built as a single-file web application, covering the technical, business, and operational knowledge needed to start a CSEB block-making business in Ghana. The training is structured for beginners and progresses to expert-level depth.

## What's inside

- **13 modules** spanning roughly 12 months at 5 hours per week
  - Module 0: Preparation & Resources (formal training providers)
  - Modules 1–12: Foundations → Soil Science → Production → Construction → Business
- **3 interactive calculators** — soil recipe, compression strength, unit economics
- **120 flashcards** with spaced repetition
- **58-term searchable glossary**
- **25-book curated reading library** across CSEB technology, faith and business, leadership, and African entrepreneurship
- **Printable study planner** with customizable hours/week
- **Multi-profile support** — multiple users can share the same device, each with separate progress
- **Final certification exam** — 30 questions, 80% to pass

## Tech notes

- Single HTML file with embedded CSS and JavaScript
- No build step, no server, no dependencies
- Progress stored in browser localStorage (per profile)
- Works completely offline once loaded
- Mobile-responsive

## Running locally

Just open `index.html` in any modern browser.
